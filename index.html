Как пользоваться сервисом

  1. Отсортируй домен в manager
  2. Нажми Zl
  3. Выбери даты, за какой период тебе необходимы логи домена
  4. Нажми CTRL + A чтобы выделить все содержимое страницы
  5. Нажми CTRL + C чтобы скопировать все содержимое страницы
  6. Нажми CTRL + V и вставь скопированные данные в поле "Входные данные"
  7. Нажми кнопку "Обработать данные"
Совет: Для быстрой очистки полей используй кнопку "Очистить".
