# DNS Logs Converter

Переводит логи изменений DNS-записей в удобный для чтения и анализа текстовый формат.


**Сервис доступен по адресу:** [https://izelenov7.github.io/dns-logs-regru/](https://izelenov7.github.io/dns-logs-regru/)

## Возможности
- Преобразование логов операций ADD/DEL с DNS-записями
- Поддержка типов записей: A, AAAA, TXT, CNAME, MX, NS, CAA
- Фильтрация по добавленным/удалённым записям
- Копирование результатов в буфер обмена
- Сохранение логов в файл с автоматическим именованием

## Как пользоваться
1. Отсортируйте домен в manager
2. Нажмите Zl
3. Выберите даты в manager, за какой период вам необходимы логи домена
4. Скопируйте все логи домена от *Дата* сверху до *}* снизу. Если вы скопируете лишние символы — ничего страшного. Главное скопируйте все содержимое таблицы
5. Вставьте скопированные данные в поле "Входные данные" и нажмите "Обработать данные"
